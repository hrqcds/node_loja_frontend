#### Passo a Passo para instalação:

1. Baixar repositório do GIT

2. Instalar dependências do NODE
   - npm install
   - yarn

3. Configurar o axios para a porta do BACKEND

<img src="./public/img/axios.png" width="500px">

4. Rodar o script do sistema:
    * npm run dev
    * yarn dev
